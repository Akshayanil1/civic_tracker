Developer Portal & Open Data API

Access anonymized civic issue data to build your own dashboards, run data science experiments, and promote civic transparency.

API Endpoint: Get Resolved Issues

This endpoint returns a list of resolved and closed civic issues, strictly stripping out Personally Identifiable Information (PII).

GET /api/method/civic_tracker.api.public.get_open_data

Parameters

Parameter Type Description
limit Integer Number of records to return (Default: 100, Max: 1000)
offset Integer Pagination offset (Default: 0)
