### Snippet for SOP

"My passion for leveraging technology to drive systemic social change culminated in the architecture and deployment of **Civic Tracker**, an open-source, headless ERP platform designed to enforce accountability in municipal governance. Recognizing the disconnect between citizen grievances and contractor execution in Indian municipalities, I engineered a highly scalable four-tier system using the Frappe Framework. 

To handle the complexity of urban data, I designed a multi-modal ingestion pipeline that accepts reports via a bespoke Web Portal, a Meta Cloud API WhatsApp bot, and high-throughput IoT webhooks connected to physical smart sensors. Crucially, to automate prioritization and prevent system abuse, I integrated an AI triage layer. Using Natural Language Processing and Vision AI, the system automatically validates image evidence, categorizes grievances, performs sentiment analysis to calculate urgency scores, and dynamically routes critical emergencies directly to district administrators. 

The most impactful feature of the platform is its financial accountability engine, which ties civic performance directly to capital. Automated background cron jobs run continuous SLA evaluations; if a contractor fails to resolve an issue within the algorithmic deadline, the system automatically triggers a financial penalty and deducts it via an ERPNext integration. By exposing this data through a PII-scrubbed Open Data API and public Ward Leaderboards, Civic Tracker transforms opaque bureaucracy into a transparent, data-driven ecosystem. 

Building this full-stack, AI-integrated civic engine from the ground up solidified my desire to pursue a Master's degree. I aim to deepen my expertise in scalable distributed systems and machine learning to build the next generation of resilient smart city infrastructures."
